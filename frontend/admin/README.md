# Amenity Booking Platform Admin Dashboard

This project is a Next.js (React + TailwindCSS) admin dashboard for managing amenities, bookings, users, and reports. The users page now connects to the FastAPI backend instead of using only mocked data.

## Features
- Sidebar navigation
- Dashboard with stats cards, bookings table, reports widget, quick actions
- Pages for Bookings, Amenities, Users, Reports, Settings
- Responsive, modern UI

## Getting Started
1. Install dependencies:
   ```sh
   npm install
   ```
2. Create `frontend/admin/.env.local` from `frontend/admin/.env.example` and set the backend values.
3. Run the development server:
   ```sh
   npm run dev
   ```
4. Open [http://localhost:3000/admin/dashboard](http://localhost:3000/admin/dashboard) in your browser.

Required local env values:

```env
ADMIN_API_BASE_URL=http://127.0.0.1:8000
ADMIN_API_KEY=your_admin_api_key_here
```

## Project Structure
- `/src/app/admin/*` - Admin dashboard routes
- `/src/components/*` - Reusable UI components
- `/src/styles/*` - Global styles

## Customization
